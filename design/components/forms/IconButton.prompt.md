**IconButton** — a square button holding one icon, for toolbars and table row actions. Always pass `label` for accessibility.

```jsx
<IconButton label="Editar" variant="ghost"><PencilIcon/></IconButton>
<IconButton label="Eliminar" variant="danger"><TrashIcon/></IconButton>
```

Variants: `ghost` · `outline` · `solid` · `danger`. Sizes: `sm` · `md` · `lg`.
